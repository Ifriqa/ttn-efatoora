# TTN e-Fatoora

TTN e-Fatoora is a Windows desktop application by **MG Nova Technology** for preparing TTN e-Fatoora workflows.

## Current availability

TTN e-Fatoora is currently available in private beta on request.

Official product page: https://ifriqa.com/products/ttn-efatoora

## Distribution

Official Windows releases are published from this repository under GitHub Releases.

Use only:

- https://ifriqa.com/products/ttn-efatoora
- https://github.com/Ifriqa/ttn-efatoora/releases

## Security and updates

- The application is distributed as a Windows MSI installer.
- Update packages are verified by the Tauri updater signature mechanism.
- Future public installers should be signed by **MG Nova Technology** using a Windows code-signing certificate.
- SHA-256 checksums are published with each release in `SHA256SUMS.txt`.
